📊 PhonePe Transaction Analysis Dashboard

📌 Project Overview

The **PhonePe Transaction Analysis Dashboard** is an interactive Power BI project that analyzes digital payment transactions to identify customer behavior, payment trends, and business performance. The dashboard helps convert raw transaction data into meaningful insights for better decision-making.

🎯 Project Objective

The main objective of this project is to:

* Analyze PhonePe transaction data using Power BI.
* Monitor transaction success and failure rates.
* Track total transaction value and transaction volume.
* Identify the most popular payment services.
* Understand customer usage patterns.
* Compare weekday and weekend transaction behavior.
* Build an interactive dashboard for business reporting.

🛠 Tools & Technologies

* Power BI
* Power Query
* DAX
* Microsoft Excel
* Data Cleaning
* Data Modeling
* Data Visualization

📈 Dashboard Features

* Executive KPI Dashboard
* Transaction Success Rate Analysis
* Total Transaction Value
* Service Category Analysis
* Top Users Analysis
* Weekday vs Weekend Comparison
* Interactive Filters and Slicers
* Dynamic Charts and KPIs

📊 Key Project Insights

* Achieved a **96.15% transaction success rate**, showing strong platform reliability.
* Processed a **528.97M total transaction value**.
* **UPI transactions** generated the highest payment volume.
* Bill Payments and Financial Services showed consistent user engagement.
* A small number of users contributed a large share of total transactions.
* Transaction activity changed between weekdays and weekends, indicating different customer usage patterns.
* Interactive filters help analyze transactions across multiple service categories.

💡 Business Value

This dashboard helps businesses:

* Monitor payment performance.
* Identify customer transaction behavior.
* Track popular digital payment services.
* Improve operational efficiency.
* Support data-driven business decisions.
* Detect payment trends for future planning.

📌 Conclusion

The PhonePe Transaction Analysis Dashboard demonstrates how Power BI can transform raw transaction data into meaningful business insights. By combining data modeling, DAX, Power Query, and interactive visualizations, the dashboard enables users to monitor performance, understand customer behavior, and make informed decisions. This project highlights practical Business Intelligence skills that can be applied to real-world financial and digital payment analytics.
